Uploads an image to Firebase storage in the path "attachedImages/productId" Returns the url of the uploaded image if operation successful and an error instead, if not.

Implementation

Future<({String? url, String? error})> uploadImageToFirebase(File imageFile, String productId) async {
  try {
    TaskSnapshot taskSnapshot = await storage.ref('attachedImages/$productId').putFile(imageFile);
    final String downloadUrl = await taskSnapshot.ref.getDownloadURL();
    return (url: downloadUrl, error: null);
  } catch (e) {
    debugPrint(e.toString());
    return (url: null, error: e.toString());
  }
}
